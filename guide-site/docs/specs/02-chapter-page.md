# 章ページ仕様

## ページ構成

各章ページは以下の構成で統一する。

```
┌─────────────────────────────────────┐
│ [ヘッダー]                           │
│ 第N章 / 全12章    読了時間    [🌙]    │
├─────────────────────────────────────┤
│                                     │
│ [パンくず] 📖 Xアルゴリズム攻略ガイド  │
│                                     │
│ ■ 章タイトル                        │
│   章の説明文                         │
│                                     │
├─────────────────────────────────────┤
│                                     │
│ [章の本文コンテンツ]                  │
│ - セクション1                        │
│ - セクション2                        │
│ - ...                               │
│                                     │
├─────────────────────────────────────┤
│                                     │
│ 📝 よくある質問 (FAQ)                │
│ - Q1: ... ▼                         │
│ - Q2: ... ▼                         │
│ - ...                               │
│                                     │
├─────────────────────────────────────┤
│                                     │
│ ✅ 理解度チェック                    │
│ - 問1: ...                          │
│   [A] [B] [C] [D]                   │
│   → 正解/不正解 + 解説              │
│ - ...                               │
│                                     │
├─────────────────────────────────────┤
│                                     │
│ [章移動ナビゲーション]               │
│ ← 前の章          次の章 →          │
│                                     │
└─────────────────────────────────────┘
```

## ヘッダー仕様

| 項目 | 内容 |
|------|------|
| 位置 | `sticky top-0` |
| 高さ | 64px |
| 背景 | `bg-background/95 backdrop-blur` |
| 左側 | 章番号表示 (例: `第1章 / 全12章`) |
| 右側 | テーマ切り替えボタン |

## 本文コンテンツ仕様

### セクション番号
- 形式: `N.M` (例: 1.1, 1.2, 1.3)
- スタイル: 章番号は青色のバッジ

### 見出しスタイル
```css
h2: text-xl font-bold border-l-4 border-primary pl-3 mb-4
h3: text-lg font-semibold mb-3
h4: text-base font-medium mb-2
```

### 特殊ボックス
- **ポイントボックス**: 黄色背景、重要ポイント表示
- **情報ボックス**: 青色背景、補足情報表示
- **警告ボックス**: 赤色背景、注意事項表示

## FAQ セクション仕様

- 最低5問を各章に含める
- アコーディオン形式で表示
- 質問をクリックで回答を展開/折りたたみ

## レスポンシブ対応

| ブレークポイント | 対応 |
|----------------|------|
| モバイル (<768px) | 1カラム、パディング縮小 |
| タブレット (768px-1024px) | サイドバー非表示 |
| デスクトップ (>1024px) | サイドバー表示、フルレイアウト |
