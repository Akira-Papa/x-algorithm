## レイアウト構成

### 共通レイアウト
- **サイドバー**: 左固定 (幅 288px / lg以上で表示)
- **メインコンテンツ**: サイドバー右側
- **モバイル対応**: ハンバーガーメニューでサイドバー表示

## ファイル構成

```
src/
├── app/
│   ├── layout.tsx          # ルートレイアウト
│   ├── page.tsx            # ホームページ
│   ├── globals.css         # グローバルスタイル
│   ├── chapters/
│   │   └── [id]/page.tsx   # 章ページ (動的ルート)
│   ├── glossary/
│   │   └── page.tsx        # 用語集ページ
│   ├── specs/
│   │   └── [slug]/page.tsx # 仕様ページ (動的ルート)
│   └── complete/
│       └── page.tsx        # 完了ページ
├── components/
│   └── ui/
│       ├── Sidebar.tsx     # サイドバー
│       ├── ChapterNav.tsx  # 章移動ナビゲーション
│       ├── Quiz.tsx        # クイズコンポーネント
│       ├── FAQ.tsx         # FAQコンポーネント
│       ├── CodeBlock.tsx   # コードブロック
│       └── ThemeToggle.tsx # テーマ切り替え
└── lib/
    ├── chapters.ts         # 章データ
    ├── glossary.ts         # 用語データ
    ├── specs.ts            # 仕様データ
    └── utils.ts            # ユーティリティ
```
