## 会議の成果サマリー

### Before → After

```
【Before: 現行カリキュラム】
- 12章構成（理論→実践の一方向）
- 専門用語が多い
- 対話形式のみ
- ターゲット不明確

【After: 改善版カリキュラム】
- 10章+補足3本（理論⇔実践の交互配置）
- 段階的用語導入
- ハイブリッド形式（解説+Q&A）
- 3ペルソナ別読み方ガイド
```
