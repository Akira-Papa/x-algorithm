## クイックスタート

### 最も重要な3つのポイント

1. **エンゲージメントが全て**
   - いいね、リプライ、リツイート、滞在時間が高スコアの鍵
   - ネガティブアクション（ブロック、報告）は大きなマイナス

2. **フォロワーとの関係が優先**
   - インネットワーク（フォロワー）は係数1.0
   - アウトオブネットワークは係数0.85（15%減）

3. **多様性と鮮度**
   - 同一著者の連続投稿は減衰される
   - 新しい投稿ほど優先される

### スコア計算式

```
Final Score = Σ (weight_i × P(action_i))

例:
= 0.5 × P(いいね) + 1.0 × P(リプライ) + ... - 2.0 × P(ブロック)
```
