## 12.2 フェーズ別ロードマップ

フォロワー数に応じた4つの成長フェーズ。各フェーズに適した戦略で着実に成長しましょう。

### Phase 1: 基盤構築（0-1,000フォロワー）

```
【Phase 1: 基盤構築期】

期間目安: 3-6ヶ月
重点: INエンゲージメントの構築

主要アクション:
├── ニッチな専門性を確立
├── 毎日1-2投稿を継続
├── フォロワーのリプライに必ず返信
└── 同分野のアカウントと積極的に交流

成功指標: エンゲージメント率 5%以上

このフェーズのポイント:
・バズを狙わない、コツコツ積み上げ
・濃い関係構築が後のOON拡散の基盤になる
・Trust Scoreを下げる行為は絶対に避ける
```

### Phase 2: 成長期（1,000-5,000フォロワー）

```
【Phase 2: 成長期】

期間目安: 6-12ヶ月
重点: OON到達の開始

主要アクション:
├── 成功パターンの横展開
├── テーマを少し広げる
├── スレッド投稿の活用
└── エンゲージメント活動を効率化

成功指標: 週100人以上のフォロワー増加

このフェーズのポイント:
・Phase 1で見つけた成功パターンを強化
・OONへの露出が始まる重要な時期
・投稿の質と量のバランスを取る
```

### Phase 3: 拡大期（5,000-10,000フォロワー）

```
【Phase 3: 拡大期】

期間目安: 6-12ヶ月
重点: 影響力の確立

主要アクション:
├── OON比率30%以上を目指す
├── コラボレーション企画
├── 動画コンテンツの導入
└── ブランディングの強化

成功指標: OONエンゲージメント比率 30%以上

このフェーズのポイント:
・影響力を持ち始める時期
・他のインフルエンサーとの連携が効果的
・動画でVQV最適化にチャレンジ
```

### Phase 4: 確立期（10,000+フォロワー）

```
【Phase 4: 確立期】

期間目安: 継続
重点: 持続可能な運用

主要アクション:
├── エンゲージメント率の維持
├── コンテンツの質を最優先
├── 分析と改善の習慣化
└── 長期的な価値提供

成功指標: エンゲージメント率 3%以上を維持

このフェーズのポイント:
・守りのフェーズ、質の維持が最重要
・チーム化や外注も検討
・長期的な視点でブランドを育てる
```

## 12.3 1日のルーティン

継続可能なペースで運用しましょう。合計約35-55分/日。

```
【朝: 分析チェック（5分）】

タイミング: 起床後、出勤前
やること:
├── 前日の投稿パフォーマンス確認
├── 通知の確認
└── トレンドのチェック

ポイント:
・サッと確認するだけ
・深い分析は週次で

【午前: 投稿作成・予約（15-30分）】

タイミング: 集中できる時間帯
やること:
├── メイン投稿の作成
├── 画像・動画の準備
└── 最適時間に予約設定

ポイント:
・投稿は午前中に作成
・予約機能を活用
・テンプレートで効率化

【午後: エンゲージメント（15分）】

タイミング: 休憩時間、移動時間
やること:
├── リプライへの返信
├── 同分野アカウントとの交流
└── 気になる投稿へのコメント

ポイント:
・スキマ時間を活用
・「いいですね」だけでなく価値あるリプライを
・1日15分程度で十分

【夜: 振り返り（5分）】

タイミング: 就寝前
やること:
├── 今日の投稿パフォーマンス
├── 明日のネタ検討
└── 気づきのメモ

ポイント:
・簡単に振り返るだけ
・良かった点と改善点を意識
```

## 12.4 チェックリスト

3種類のチェックリストで品質を担保します。

### 投稿前チェックリスト

```
【投稿前チェックリスト】

□ ターゲット読者を明確にした
  └─ 誰に向けた投稿か確認

□ 価値を提供している
  └─ 読者が得られるメリットを確認

□ 冒頭で興味を引ける
  └─ 最初の1行で続きを読みたくなるか

□ 外部リンクは必要最小限
  └─ リンクなしで完結できないか再検討

□ 画像・動画は適切
  └─ 視覚的要素が内容を強化しているか

□ 投稿時間は最適
  └─ ターゲットがアクティブな時間か

すべてチェックできたら投稿OK
```

### 週次レビューチェックリスト

```
【週次レビューチェックリスト】

所要時間: 30分
タイミング: 週末 or 週初め

□ 週間インプレッション合計を記録
  └─ 前週との比較も

□ エンゲージメント率を計算
  └─ (いいね+RT+リプライ)/インプレッション

□ フォロワー純増数を確認
  └─ 新規フォロー - アンフォロー

□ TOP3投稿を分析
  └─ 成功要因を特定

□ WORST投稿を分析
  └─ 失敗要因を特定

□ 来週の投稿計画を立てる
  └─ テーマとスケジュール

すべてチェックできたら週次分析完了
```

### 月次振り返りチェックリスト

```
【月次振り返りチェックリスト】

所要時間: 1時間
タイミング: 月初め

□ 月間KPI達成度を評価
  └─ 目標との差分を確認

□ 成功パターンを文書化
  └─ 再現可能な形で記録

□ 失敗パターンをブラックリスト化
  └─ 避けるべき行動を明確に

□ 来月の目標を設定
  └─ 数値で具体的に

□ 戦略の修正点を洗い出す
  └─ PDCAのAct

□ フェーズの進捗を確認
  └─ 次のフェーズへの準備

すべてチェックできたら月次振り返り完了
```
