# 第2章：エンゲージメント19種類の完全解説

## 2.1 エンゲージメントとは何か

エンゲージメントとは、ユーザーが投稿に対して行うあらゆる反応のことです。いいね、リプライ、リポストといった明示的なアクションから、投稿をどれだけの時間見ていたかという暗黙的な行動まで、すべてがエンゲージメントとして計測されています。

Xのアルゴリズムは、各エンゲージメントに**重み（weight）**を設定しています。この重みは、そのエンゲージメントがどれだけ「価値がある」とアルゴリズムが判断しているかを示します。

重要なのは、すべてのエンゲージメントが同じ価値ではないということです。いいね1回とリプライ1回では、アルゴリズム的な価値が**27倍**も異なります。

## 2.2 ポジティブエンゲージメント15種類

ソースコード `weighted_scorer.rs` から抽出した、ポジティブエンゲージメント15種類とその重みを解説します。

### 基本インタラクション（日常的に発生）

```rust
【基本インタラクション 4種類】

favorited: 0.5
// いいね - 最も頻繁に行われるが、重みは低い
// 気軽にできるため、深いエンゲージメントとは見なされない

retweeted: 1.0
// リポスト - いいねの2倍の重み
// 「自分のフォロワーにも見せたい」という意思表示

replied: 13.5
// リプライ - 最も重要なエンゲージメント
// 会話が生まれる = 滞在時間増加につながる

quoted: 1.1
// 引用リポスト - リポストよりわずかに高い
// 自分の意見を付加している = より深い関与
```

**戦略的含意**: リプライの重みが13.5という異常に高い値であることに注目してください。これは、Xが「会話を生み出すコンテンツ」を最も高く評価していることを意味します。

### 深いエンゲージメント（高価値）

```rust
【深いエンゲージメント 5種類】

profile_clicked: 12.0
// プロフィールクリック - リプライに次ぐ高さ
// 「この人をもっと知りたい」という強い興味

detail_expanded: 0.3
// 詳細展開 - 投稿をクリックして全文を見る
// 興味はあるが、アクションまでは至っていない

video_playback_50: 0.005
// 動画50%視聴 - 非常に低い重み
// 単なる視聴より、VQV（後述）が重要

video_quality_view: 1.5
// VQV（Video Quality View）- 動画の質の高い視聴
// 10秒以上、音声ON、スクロール通過でない視聴

dwell_time_ms: 動的計算
// 滞在時間 - 投稿を見ていた時間（ミリ秒）
// 長いほど高評価だが、上限あり
```

**戦略的含意**: プロフィールクリック（12.0）は、いいね（0.5）の24倍の価値があります。投稿内でプロフィールへの興味を喚起することは、極めて効果的な戦略です。

### フォロー関連（最高価値）

```rust
【フォロー関連 3種類】

followed: 4.0
// フォロー - 継続的な関係構築の始まり
// 一度のフォローがいいね8回分の価値

unfollowed_later: 追跡対象
// 後のアンフォロー - ネガティブシグナルとして追跡
// フォロー後すぐのアンフォローはペナルティ要因

notification_enabled: 2.0
// 通知オン - 「この人の投稿を見逃したくない」
// フォローの半分の重みだが、非常に強い興味表示
```

**戦略的含意**: フォロー（4.0）と通知オン（2.0）は、長期的な関係性を示す指標です。一時的なバズより、フォロワーとの継続的な関係構築が重要です。

### その他のポジティブエンゲージメント

```rust
【その他 3種類】

link_clicked: 1.1
// リンククリック - 外部リンクへのクリック
// 投稿の情報に価値を感じた証拠

bookmark_added: 1.0
// ブックマーク - 「後で見返したい」
// リポストと同じ重みだが、非公開の行動

shared_externally: 1.2
// 外部共有 - X外へのシェア
// Xのコンテンツを外部に広める価値
```

## 2.3 ネガティブエンゲージメント4種類

ネガティブエンゲージメントは、投稿の評価を大きく下げる要因です。

```rust
【ネガティブエンゲージメント 4種類】

reported: -10.0
// 通報 - 最も重いペナルティ
// 1件の通報で、いいね20回分の価値が消える

blocked: -10.0
// ブロック - 通報と同等のペナルティ
// この投稿主との関係を完全に断つ意思表示

muted: -5.0
// ミュート - ブロックより軽いが重大
// 「見たくないが、関係は断たない」

not_interested: -1.5
// 興味なし - 最も軽いネガティブシグナル
// 「おすすめ」でのフィードバック
```

**重要な理解**: ネガティブエンゲージメントの影響は非常に大きいです。1件の通報やブロック（-10.0）は、20いいね分の価値を消し去ります。攻撃的・不快なコンテンツは、たとえ「バズ」しても、最終的にはアカウント全体の評価を下げます。

## 2.4 重みの戦略的含意

19種類の重みを戦略的な観点から整理します。

```
【重み順位表】

順位  エンゲージメント      重み    いいね比
─────────────────────────────────────────
1位   リプライ            13.5    27倍
2位   プロフィールクリック  12.0    24倍
3位   フォロー             4.0     8倍
4位   通知オン             2.0     4倍
5位   VQV動画視聴          1.5     3倍
6位   外部共有             1.2    2.4倍
7位   引用RT               1.1    2.2倍
8位   リンククリック        1.1    2.2倍
9位   リポスト             1.0     2倍
10位  ブックマーク          1.0     2倍
11位  いいね               0.5     1倍
```

### 戦略1：リプライを誘発する

リプライ（13.5）は最も価値が高いエンゲージメントです。リプライを誘発するための技術：

```
【リプライ誘発テクニック】

質問で終わる
├── 「みなさんはどう思いますか？」
├── 「あなたの経験を教えてください」
└── 「どちらを選びますか？」

意見が分かれるテーマ
├── 業界の常識への問いかけ
├── 2つの選択肢の提示
└── controversial（議論を呼ぶ）だが攻撃的でない内容

体験の共有を促す
├── 「同じ経験をした人いますか？」
├── 「これで困った経験ありませんか？」
└── 「成功した方法を教えてください」
```

### 戦略2：プロフィールへの誘導

プロフィールクリック（12.0）は、リプライに次ぐ高価値です。

```
【プロフィール誘導テクニック】

実績の断片を見せる
├── 「3年で0→10万フォロワーを達成した方法」
├── 「月100万円を稼いだ具体的な手順」
└── → 「詳細はプロフィールで」

希少情報の予告
├── 「業界では知られていない裏話」
├── 「公開してない情報をプロフで共有中」
└── → 好奇心でクリックを誘発

専門性のアピール
├── 「10年間この分野を研究してきました」
├── 「1000社以上をコンサルした経験から」
└── → 「どんな人か」を知りたくなる
```

### 戦略3：ネガティブを避ける

ネガティブエンゲージメントの影響は壊滅的です。

```
【絶対に避けるべきこと】

通報・ブロックを招く行為
├── 攻撃的・差別的な表現
├── 虚偽の情報の拡散
├── スパム的な大量投稿
└── 他者への誹謗中傷

ミュートを招く行為
├── 同じ内容の繰り返し投稿
├── 過度な自己宣伝
├── フォロワーの興味と乖離した内容
└── 政治的・宗教的な過激発言
```

## 2.5 よくある誤解と正解

### 誤解1：「いいねを大量に獲得すれば拡散される」

**正解**: いいね（0.5）よりリプライ（13.5）の方が27倍重要です。100いいねより4リプライの方がアルゴリズム的には高評価。「いいね数」を追いかけるより、「会話を生む投稿」を意識しましょう。

### 誤解2：「バズれば勝ち」

**正解**: バズった投稿でも、通報やブロックが発生していれば、アカウント全体の評価が下がります。1件の通報（-10.0）は、20いいね分のプラスを消し去ります。持続可能な成長には、ネガティブを避けることが不可欠です。

### 誤解3：「リポストは最も重要」

**正解**: リポスト（1.0）はいいね（0.5）の2倍ですが、リプライ（13.5）やプロフィールクリック（12.0）と比べると重要度は低いです。リポストを狙うより、リプライやプロフィール訪問を誘発する方が効果的です。

### 誤解4：「動画は再生数が大事」

**正解**: 単純な動画再生（50%視聴で0.005）より、VQV（1.5）の方が300倍重要です。VQV（Video Quality View）は、10秒以上・音声ON・スクロール通過でない視聴を意味します。再生「数」より、再生「質」が重要です。
