## 本書について

本書は、Xのオープンソース推薦アルゴリズムを徹底解析し、その仕組みを実践的な戦略に落とし込んだ完全ガイドです。

2023年にX（旧Twitter）が推薦アルゴリズムをオープンソース化したことで、私たちは初めて「なぜある投稿が広く表示され、別の投稿が埋もれるのか」を技術的に理解できるようになりました。

本書では、実際のソースコード（Rust/Python）を解析し、19種類のエンゲージメント重み、12種類のフィルター、機械学習モデルの内部構造まで、すべてを明らかにします。

## 本書の特徴

### 1. ソースコードに基づく正確性

すべての数値と仕組みは、実際のソースコードから抽出しています。

```
weighted_scorer.rs → エンゲージメント重み
recsys_model.py → MLモデル構造
oon_scorer.rs → フォロワー外ペナルティ
author_diversity_scorer.rs → 著者多様性減衰
```

### 2. 実践的な戦略への落とし込み

技術的な解説だけでなく、「今日から何をすべきか」を明確にしています。

### 3. 段階的な理解設計

基礎→技術詳解→実践応用の流れで、無理なく深い理解に到達できます。
