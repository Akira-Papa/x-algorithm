# 第7章：IN vs OON - フォロワー戦略の核心

## 7.1 INとOONとは何か

Xのアルゴリズムは、投稿の閲覧者を2つのカテゴリに分類しています。

```
【IN と OON の定義】

IN（In-Network）
└─ あなたをフォローしているユーザー
   └─ あなたの投稿がタイムラインに「自然に」流れる
   └─ フォロー関係に基づく直接的な到達

OON（Out-of-Network）
└─ あなたをフォローしていないユーザー
   └─ あなたの投稿が「おすすめ」経由で届く
   └─ アルゴリズムによる推薦に基づく到達
```

この区分は、単なる分類ではありません。アルゴリズムの処理において、**明確な差別化**が行われています。

```
【IN と OON の扱いの違い】

                    IN              OON
─────────────────────────────────────────
スコア係数         1.0             0.85
到達経路           タイムライン     おすすめ
必要スコア         標準            標準の約1.18倍
信頼度             高い            要証明
```

## 7.2 15%ペナルティの重大性

OON Scorerによる15%ペナルティは、見た目以上に大きな影響があります。

### 数値で見る影響

```
【15%ペナルティの影響計算】

同じPhoenix Score 100の投稿があるとして:

INユーザーへの最終スコア:
100 × 1.0 = 100

OONユーザーへの最終スコア:
100 × 0.85 = 85

差分: 15ポイント（15%減）
```

しかし、この15%は「絶対値」の話です。**相対的な競争**で考えると、影響はもっと大きくなります。

```
【相対的な影響】

タイムラインに表示される候補が100件あり、
上位10件だけが実際に表示されるとする。

あなたの投稿のスコアが85（OON）の場合:
- スコア84以下の投稿には勝つ
- スコア86以上の投稿には負ける

あなたの投稿のスコアが100（IN）の場合:
- スコア99以下の投稿には勝つ
- スコア101以上の投稿には負ける

→ 競争相手が15%多くなる（実質的な不利）
```

### 累積効果

さらに、他のスコアラーとの掛け算で累積します。

```
【累積効果の例】

Phoenix Score: 100
OON Score: 0.85（非フォロワー）
Author Diversity: 0.95（2投稿目）

最終スコア = 100 × 0.85 × 0.95 = 80.75

vs

Phoenix Score: 100
IN Score: 1.0（フォロワー）
Author Diversity: 1.0（初投稿）

最終スコア = 100 × 1.0 × 1.0 = 100

差分: 約20%の差
```

## 7.3 INファースト戦略

OONへの到達を狙う前に、まずINでの成功を確保する戦略です。

### なぜINファーストか

```
【INファーストの論理】

1. INでの反応がOON拡散の「燃料」
   ├── INで高エンゲージメント → Phoenix Scoreが上昇
   ├── 高いPhoenix Score → OONでもペナルティを相殺
   └── OON到達 → 新規フォロワー獲得 → IN拡大

2. INは「確実な到達」
   ├── フォロワーのタイムラインに優先表示
   ├── ペナルティなしでスコア評価
   └── 反応を得やすい（既にあなたを認知）

3. OONは「不確実な到達」
   ├── 15%ペナルティで不利な戦い
   ├── あなたを知らない相手に届ける必要
   └── より高いスコアが必要
```

### INファーストの実践

```
【INファースト 実践ステップ】

Step 1: フォロワーとの関係を深める
├── フォロワーのリプライに必ず返信
├── フォロワーの投稿にいいね・リプライ
└── フォロワーの興味に合った投稿を心がける

Step 2: INでの反応を最大化
├── 投稿時間をフォロワーの活動時間に合わせる
├── リプライを誘発する質問で終わる
└── プロフィールへの誘導を入れる

Step 3: INの反応を観察
├── どの投稿がリプライを集めたか
├── どの投稿がプロフィールクリックを生んだか
└── 成功パターンを特定する

Step 4: 成功パターンを継続
├── 反応が良かった形式を繰り返す
├── 反応が悪かった形式は避ける
└── 継続的な改善サイクル
```

## 7.4 2段階成長戦略

フォロワー数に応じた戦略の切り替えが必要です。

### Phase 1: INファースト期（〜1,000フォロワー）

```
【Phase 1: 〜1,000フォロワー】

目標:
├── フォロワーとの強い関係構築
├── 高エンゲージメント率の維持
└── 成功する投稿パターンの確立

重視する指標:
├── エンゲージメント率（5%以上目標）
├── リプライ率（1%以上目標）
└── フォロワー1人あたりの価値

やるべきこと:
├── 毎日1-2投稿（質重視）
├── すべてのリプライに返信
├── ターゲット層のアカウントと積極交流
└── ニッチを絞った専門性の確立

避けるべきこと:
├── バズ狙いの汎用的なコンテンツ
├── フォロワー購入やスパム的行為
└── 投稿頻度の無理な増加
```

### Phase 2: OON展開期（1,000フォロワー〜）

```
【Phase 2: 1,000フォロワー〜】

変化:
├── INでの反応がOON拡散を生み始める
├── 「おすすめ」タブでの露出が増加
└── 新規フォロワー獲得サイクルが回り始める

目標:
├── OON到達の拡大
├── IN→OON→IN（新規フォロワー）のサイクル確立
└── フォロワー増加の加速

重視する指標:
├── OONインプレッション比率
├── 新規フォロワー獲得数
└── フォロー転換率（見た人がフォローする率）

やるべきこと:
├── INでの成功パターンを維持
├── より広いテーマへの段階的拡張
├── OON向けの「入口」コンテンツ作成
└── プロフィールの最適化（初見でフォローしたくなる）

避けるべきこと:
├── INを無視したOON狙いの過度な戦略
├── ニッチからの急激な離脱
└── フォロワーとの関係の希薄化
```

## 7.5 フォロワー数別の戦略調整

より詳細なフォロワー数帯別の戦略です。

### 0-100フォロワー

```
【0-100フォロワー】

状況:
├── INが非常に少ない
├── アルゴリズムがあなたを「学習中」
└── OON到達はほぼ期待できない

戦略:
├── とにかくフォロワーを増やす（質重視）
├── ターゲット層のアカウントにリプライ
├── 自分の専門性を明確にする投稿
└── プロフィールを最適化

重要指標:
├── フォロワー増加数（週10人以上目標）
└── リプライしたアカウントからのフォローバック率
```

### 100-1,000フォロワー

```
【100-1,000フォロワー】

状況:
├── INが形成され始める
├── 投稿への反応が安定し始める
└── OON到達が時々発生

戦略:
├── INエンゲージメント率を最大化
├── 成功する投稿パターンを特定
├── 投稿の質と頻度のバランスを探る
└── フォロワーとの関係を深める

重要指標:
├── エンゲージメント率（5%以上）
├── リプライ率（1%以上）
└── 週次フォロワー増加数
```

### 1,000-10,000フォロワー

```
【1,000-10,000フォロワー】

状況:
├── INが十分な規模に
├── OON到達が定常的に発生
└── 成長サイクルが回り始める

戦略:
├── INを維持しつつOON拡大
├── コンテンツの種類を少し広げる
├── 「入口」コンテンツと「深い」コンテンツの使い分け
└── フォロー転換率の最適化

重要指標:
├── OONインプレッション比率（30%以上）
├── 日次フォロワー増加数
└── フォロー転換率
```

### 10,000+フォロワー

```
【10,000+フォロワー】

状況:
├── 影響力のあるアカウント
├── OON到達が主要な成長エンジン
└── バズりやすい状態

戦略:
├── コンテンツ品質の維持
├── フォロワーとの関係の維持
├── 影響力の責任ある活用
└── 新規層へのリーチ拡大

重要指標:
├── エンゲージメント率の維持（3%以上）
├── ブランドとしての一貫性
└── 影響の質（ポジティブなネガティブか）
```
